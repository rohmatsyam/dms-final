## How to run the code
- git clone
- cd to project folder
- cp .env.example `.env`
- open .env and update DB_DATABASE
- run : `php artisan key:generate`
- run : `composer install`
- run : `npm install && npm run dev`
- run : `php artisan migrate:fresh --seed`
- run : `php artisan serve`
